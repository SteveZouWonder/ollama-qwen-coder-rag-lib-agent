# 更新日志

本文档记录了项目的主要版本变更历史。

## [v4.2.1] - 2026-06-22

### 修复问题
- **网络搜索优化**: 修复DuckDuckGo搜索包警告和稳定性问题
  - 将`duckduckgo_search`包更新为新的`ddgs`包，消除RuntimeWarning
  - 添加Wikipedia搜索引擎作为备用，提高搜索可靠性
  - 改进中英文查询转换策略，提升搜索效果
  - 实现自动降级机制：DuckDuckGo失败时自动切换到Wikipedia
  - 优化查询提取逻辑，智能移除前缀后缀，生成更有效的搜索词
- **搜索结果准确性修复**: 修复版本查询返回过时信息的问题
  - 改进版本查询策略，使用更精确的搜索词（如"Java SE latest version"）
  - 添加官方页面内容提取功能，获取最新版本号信息
  - 实现智能缓存清除，避免过时信息影响结果
  - 新增`clear_web_search_cache`工具，支持手动清除搜索缓存

### 技术改进
- **搜索引擎**: 新增`WikipediaSearchEngine`类，使用Wikipedia官方API
- **查询处理**: 改进中英文查询转换，支持技术术语识别和翻译
- **版本查询**: 针对版本查询添加专门的搜索策略和备用查询
- **内容提取**: 集成网页内容提取功能，自动获取官方页面的详细版本信息
- **缓存管理**: 添加缓存清除工具，支持手动清理过时搜索结果
- **错误处理**: 增强搜索错误处理和日志记录
- **依赖更新**: requirements.txt中将duckduckgo-search更新为ddgs>=5.3.0

### 系统提示更新
- 强制AI使用网络搜索工具，禁止回答"无法访问互联网"
- 添加网络搜索强制性规则和使用指南
- 新增缓存清除策略，当结果可能过时时建议清除缓存

## [v4.2.0] - 2026-06-17

### 新增功能
- **智能命令推荐系统**: 添加基于上下文的智能命令推荐功能
  - 工作流分析：根据用户当前工作流程推荐下一步操作
  - 状态感知：基于系统状态推荐相关命令
  - 历史分析：根据用户历史使用模式推荐个性化命令
  - 用户学习：学习用户偏好，动态调整推荐策略
  - CLI集成：在命令行界面显示推荐，支持Tab键快速执行

### 技术实现
- **新增模块**: `src/command_recommender/` 推荐引擎模块
  - 配置管理（config.py）
  - 类型定义（types.py）
  - 工作流分析器（workflow.py）
  - 状态分析器（state.py）
  - 历史分析器（history.py）
  - 学习引擎（learning.py）
  - 显示格式化器（display.py）
  - 上下文管理器（context.py）
  - 推荐引擎核心（engine.py）
- **测试覆盖**: 新增 `tests/test_command_recommender/` 测试套件
  - 165个测试通过，8个跳过
  - 代码覆盖率69%（目标95%）

### 系统集成
- **CLI集成**: 扩展 `query_interface.py` 支持推荐功能
- **桌面应用（托盘进程）**: 不集成推荐功能 - 推荐系统仅在CLI中实现
- **配置管理**: 支持推荐系统配置选项

### 文档更新
- **技术文档**: 新增 `docs/implemented-features/f4-command-recommender/` 完整技术文档
  - README.md: 概述文档
  - OVERVIEW.md: 功能概述与技术选型
  - ARCHITECTURE.md: 系统架构设计
  - IMPLEMENTATION.md: 实现指南
  - TESTING.md: 测试策略
  - IMPLEMENTATION_SUMMARY.md: 实施总结
- **项目配置**: 更新 `.devin/AGENTS.md` 和 `.devin/SYSTEM_PROMPT.md`
- **主文档**: 更新 README.md、TESTING.md、TUTORIAL.md

### 性能优化
- **推荐响应时间**: < 100ms
- **内存占用**: < 50MB
- **计算优化**: 缓存、增量计算、懒加载
- **I/O优化**: 异步写入、批量操作

### 质量改进
- **测试覆盖**: 新增165个单元测试

## 版本说明

### 版本号规则
- 主版本号：重大架构变更或不兼容更新
- 次版本号：新功能添加或重要改进
- 修订版本号：Bug修复和小幅改进

### 更新频率
- 主版本：按需发布
- 次版本：功能完成后发布
- 修订版本：Bug修复时发布

### 兼容性
- 主版本升级可能包含不兼容变更
- 次版本升级保持向后兼容
- 修订版本升级完全兼容
