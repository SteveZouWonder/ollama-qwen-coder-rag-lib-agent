## 🎯 AI执行流程（强制）

### 新的执行流程
```
1. 读取全局配置
   - ~/.config/devin/AI_DEBUGGING_WORKFLOW.md
   - ~/.config/devin/AGENTS.md

2. 读取项目配置
   - .devin/AGENTS.md

3. 读取系统提示
   - .devin/SYSTEM_PROMPT.md

4. 读取项目知识库提示词（根据任务类型）
   - 核心提示词：PROJECT_OVERVIEW + ARCHITECTURE + CODE_STANDARDS + WORKFLOWS
   - 模块提示词：MODULE_GUIDES, TESTING_GUIDELINES, TRAP_AVOIDANCE
   - 工具提示词：TOOL_USAGE

5. 创建任务追踪
   - 使用 todo_write 工具

6. 执行任务
   - 遵循标准流程
   - 更新任务状态
```

## 🚀 使用验证

### AI执行验证
AI在执行任务时将按照以下流程操作：

1. **自动读取系统提示**
   - 检查SYSTEM_PROMPT.md中的强制步骤
   - 确认需要读取AI_KNOWLEDGE_BASE中的提示词

2. **按任务类型加载提示词**
   - 代码修改任务：加载核心提示词 + 相关模块详解
   - 测试任务：加载测试指南 + 相关模块详解
   - 工具使用任务：加载工具使用指南

3. **遵循标准流程**
   - 使用todo_write追踪任务
   - 按照WORKFLOWS.md中的流程执行
   - 遵循CODE_STANDARDS.md中的规范

4. **避免常见陷阱**
   - 参考TRAP_AVOIDANCE.md
   - 避免状态污染、测试隔离等问题
