# 项目概览 - 智能文档+代码助手 v4.1

## 🎯 核心定位

**项目名称**: 智能文档+代码助手 v4.1  
**核心技术**: Ollama qwen2.5-coder:7b + RAG + ReAct Agent  
**应用场景**: 文档知识库检索 + 代码自动化操作 + 多Agent协作

这是一个融合型AI助手项目，集成了多种AI能力：
- 📚 RAG知识库检索 - 基于用户文档的智能问答
- 🤖 ReAct Agent - 代码操作、文件管理、命令执行
- 👥 多Agent协作 - 复杂任务分解和并行处理
- 💡 智能命令推荐 - 基于上下文的命令建议
- 🌐 网络搜索 - 实时信息获取和网页内容提取

## 🏗️ 融合架构

```
统一CLI交互层 (/ask, /agent, /multi-agent, 智能命令推荐)
        │
        ├───→ RAG知识库引擎 (LlamaIndex + ChromaDB)
│               ├── 文档检索
│               ├── 语义搜索
│               └── 来源追溯
│
├───→ Agent系统 (ReAct引擎 + 工具链)
│       ├── Multi-Agent Orchestrator
│       ├── MasterAgent (主控)
│       ├── CodeAgent (代码)
│       ├── RAGAgent (知识库)
│       ├── TestAgent (测试)
│       ├── DocAgent (文档)
│       └── AuditAgent (审计)
│
└───→ 支持系统
        ├── 命令推荐系统
        ├── 网络搜索模块
        ├── 会话管理
        ├── OCR处理
        ├── Git集成
        └── 知识图谱
```

## 📦 核心模块职责

### 1. RAG引擎 (`rag_engine.py`)
- **职责**: 知识库文档检索和语义搜索
- **关键功能**:
  - 文档向量化存储
  - 语义相似度计算
  - 智能检索和排序
  - 来源追溯
- **存储**: ChromaDB向量数据库
- **状态**: 持久化索引，支持快照管理

### 2. Agent系统 (`agents/`, `agent_orchestrator.py`)
- **职责**: 任务执行和智能推理
- **关键功能**:
  - ReAct推理循环
  - 工具调用和结果处理
  - 多Agent协作
  - 任务分解和调度
- **通信**: MessageBus消息总线
- **状态**: 无状态设计，每次任务独立执行

### 3. 命令推荐系统 (`command_recommender/`)
- **职责**: 智能命令建议和用户学习
- **关键功能**:
  - 上下文感知推荐
  - 工作流分析
  - 历史学习
  - 用户偏好管理
- **存储**: 本地历史和偏好文件
- **更新**: 实时学习和优化

### 4. 网络搜索模块 (`web_search/`)
- **职责**: 实时网络信息获取
- **关键功能**:
  - DuckDuckGo搜索集成
  - 搜索结果缓存
  - 内容提取和清理
  - 结果去重和排序
- **引擎**: DuckDuckGo Search API
- **缓存**: 本地文件缓存

### 5. 会话管理 (`session_manager.py`)
- **职责**: 对话状态持久化
- **关键功能**:
  - 会话创建和切换
  - 对话历史压缩
  - 会话归档和删除
  - 状态恢复
- **存储**: JSON文件
- **生命周期**: 用户手动管理

### 6. OCR处理 (`ocr_processor/`)
- **职责**: 图片和扫描版PDF文字识别
- **关键功能**:
  - 多引擎支持 (PaddleOCR, Tesseract)
  - 图片预处理
  - 智能缓存
  - 批量处理
- **支持**: PNG, JPG, PDF等格式
- **兼容性**: Python 3.13推荐使用Tesseract

### 7. Git集成 (`git_integration/`)
- **职责**: Git版本控制集成
- **关键功能**:
  - Git历史分析
  - 提交信息生成
  - 变更追踪
  - 代码审查辅助
- **实现**: subprocess调用系统git命令
- **依赖**: 系统git命令，非gitpython库

### 8. 知识图谱 (`knowledge_graph/`)
- **职责**: 文档语义关系构建
- **关键功能**:
  - 实体提取
  - 关系构建
  - 图谱查询
  - 语义推理
- **存储**: NetworkX图结构
- **状态**: 内存中，支持导出

### 9. 工具链 (`agent_tools.py`)
- **职责**: AI可调用的工具集合
- **关键功能**:
  - 文件操作工具
  - 命令执行工具
  - RAG检索工具
  - 网络搜索工具
  - 安全检查机制
- **注册**: ToolRegistry统一管理
- **安全**: CommandSafetyChecker危险命令检测

### 10. 查询接口 (`query_interface.py`)
- **职责**: 统一CLI交互入口
- **关键功能**:
  - 命令解析和分发
  - 进度显示
  - 错误处理
  - 用户交互
- **命令**: /ask, /agent, /multi-agent, /file, /exec等

## 🔄 数据流设计

### 知识库检索流程
```
用户提问 → 向量化 → ChromaDB检索 → 结果排序 → LLM生成回答
           ↓
      nomic-embed-text
```

### Agent任务执行流程
```
用户任务 → 任务分析 → Agent选择 → 工具调用 → 结果整合 → 最终回答
            ↓
        Multi-Agent调度
```

### 网络搜索流程
```
关键词检测 → DuckDuckGo搜索 → 结果处理 → 缓存 → 整合到回答
```
