3. **系统提示强化**

1. **使用 read_system_prompt 工具**读取系统提示文件
2. **读取全局工作流程** `~/.config/devin/AI_DEBUGGING_WORKFLOW.md`
3. **读取全局质量标准** `~/.config/devin/AGENTS.md`
4. **读取项目特定配置** `.devin/AGENTS.md`
5. **使用 todo_write 创建任务列表**
6. **按照要求执行任务**
7. **更新任务进度**
8. **确保所有步骤完成**
